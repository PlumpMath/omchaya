# Omchaya

A [ClojureScript](https://github.com/clojure/clojurescript) + [Om](https://github.com/swannodette/om) frontend for [Kandan](http://kandanapp.com/). An example app that's actually useful.

## Setup

Compile the files

    git clone
    cd omchaya
    lein cljsbuild auto

Serve the assets:

    python -m SimpleHTTPServer 9000
